# Project Overview

Build a refined, minimalist-elegant website for Savannah Villegas, a Tennessee-based videographer and social-first video editor.

The site should position Savannah as:

- calm
- trusted
- artistic
- established
- emotionally intelligent
- premium without being flashy

This is NOT a creator-economy “hustle” brand.

It should feel more like:

- editorial
- boutique production studio
- cinematic storyteller
- luxury hospitality brand
- Italian lifestyle publication
- modern creative house

The emotional feeling should be:

> “This person creates thoughtful content that elevates brands.”
> 

The site should feel:

- spacious
- breathable
- elegant
- feminine but grounded
- cinematic
- calm
- human
- warm
- highly intentional

Avoid:

- aggressive social media energy
- neon creator aesthetics
- trendy startup UI
- loud gradients
- excessive animations
- tech agency feeling
- generic freelancer portfolio feel

The Brunello Cucinelli reference aesthetic is a major inspiration:

- restraint
- whitespace
- editorial typography
- architectural layout
- emotional calm
- timeless palette
- premium simplicity

# Sorted Updates / operational CMS notes

Clients do not manage the site through WordPress-style dashboards, page builders, or admin panels. They request updates through `/sortedupdates` (conversational assistant UI).

## Content architecture guidance

Structure content cleanly into reusable blocks:

- hero sections
- service blocks
- reel galleries
- testimonials
- process sections
- CTA sections
- portfolio entries

Avoid deeply hardcoded page layouts where possible.

## Future expansion (keep architecture extensible)

The `/sortedupdates` system may eventually support:

- AI-assisted updates
- portfolio uploads + image optimization
- reel publishing
- CRM integration
- booking management
- analytics summaries

So the frontend architecture should remain modular and API-friendly.

## User experience philosophy

The client experience should feel:

> “I just message what I want changed and it gets sorted.”
